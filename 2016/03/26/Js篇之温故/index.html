Js篇之温习

发表于 2016-03-26   |   分类于 Javascript   |  

全能人气小王子之Javascript

从 2013 至 2016 年，JS 已四年蝉联最热门技术。数据来自StackOverflow 2016 年开发者调查，详见JavaScript 是世界上最好的语言？

而作为前端没有理由不学好学透Js。虽然干前端有段时间了，但感觉一直没怎么深入，所以就从温习开始吧！

一. Js 基础

1. 语言特点

  • 解释型语言
  • 类似于C和Java的语法结构
  • 动态语言
  • 基于原型的面向对象
  • 字面量的表现能力
  • 函数式编程

2. 变量

  • 使用关键字 var 声明变量

    var abc; // 1.只声明但未赋值的变量，值为undefined 2.使用var 声明的变量为显示声明，如在函数内部声明则为局部变量，在外部声明即为全局变量

  • 通过赋值运算符 = 为其赋值

    abc = 'abc111' // 1.把字符串abc111赋值给变量abc 2.变量可为数字，字符串，布尔值, 数组, 对象, null, undefind

  • 也可以在声明变量的同时为其赋值

    var abc = 'abc'

  • 省略var声明变量

    ` abb = 25; // 隐式生命变量，这样声明的变量为全局变量，即使写在函数内部也为全局变量。即没通过var声明的变量都为隐式全局变量。我们应该尽可能的避免使用全局变量，杜绝使用省略var的声明方式，不管是局部还是全局都用显示声明。

3. 常量

  • 使用关键字 const 声明常量

    const abc = 7; // 1.如果我们只声明为赋值的话同变量一样为undefind，2.如果给一个常量再次赋值，这个常量值不会发生改变，

4. 函数

  • 函数声明与调用

    function sun(a,b){   //声明，sun为函数名，a,b为参数，参数可为N多个
   return Number(a)+Number(b)     //函数体  ruturn出a,b相加的结果，Number()把a的值转换为数字
}
sun(1,2)    //调用

  • 自执行匿名函数写法

     （function(){
    alert('asd')
 }())    //jslint推荐写法，方便阅读

（function(){
    alert('sad')
 })()

5. 对象

  • 可以通过对象字面量来生成一个对象，对象字面量表达式由大括号{}括起，内部有属性名和属性值

    var obj = {属性名：属性值，属性名：属性值，。。。}     // 1.属性值可为任意类型。2.可以通过obj.属性名 来访问对应属性值也可通过obj['属性名']的方式来访问对应属性值

6. new 表达式

  • js中new表达式的作用是生成一个对象

    var obj = new object();  等同于  var obj = {};
